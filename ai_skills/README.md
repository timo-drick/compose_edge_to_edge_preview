# AI Skills

This folder contains reusable skills that AI agents can apply when working with this repository or when integrating its libraries into other projects.

## Available skills

- [`integrate-edge-to-edge-preview-lib`](integrate-edge-to-edge-preview-lib/SKILL.md): Integrate `edge_to_edge_preview_lib` (`de.drick.compose:edge-to-edge-preview`) into Android or Kotlin Multiplatform Compose projects.
- [`integrate-edge-to-edge-test-lib`](integrate-edge-to-edge-test-lib/SKILL.md): Integrate `edge_to_edge_test_lib` (`de.drick.compose:edge-to-edge-test`) into Android Compose test pipelines (instrumented and Robolectric).
- [`integrate-edge-to-edge-preview-check-lib`](integrate-edge-to-edge-preview-check-lib/SKILL.md): Integrate `edge_to_edge_preview_check_lib` (`de.drick.compose:edge-to-edge-preview-check`) to highlight window-insets overlap issues in Compose previews.
